Portfolio 企业网格卡片 · 迭代汇总

把「被投企业 / 企业案例」网格从纯文字卡改造成图片卡的全过程。卡片样式起源于 team grid， 再应用到 portfolio。下面按迭代顺序排列，每段嵌入当时的探讨页、可单独打开看完整交互（hover 看磨砂上展 / logo 浮现）。

R1 · 卡片样式（选 B：姓名+副标题常显） → R2 · 磨砂高度 36% → R3 · 图片 + 文字 混合网格 → R4 · logo 直接叠加 · hover 浮现 → ★ 选定版
R1 卡片样式（team grid 起源） 基础版 单独打开 ↗

用首页「代表项目」.pcard 图片卡格式重写网格卡，对比 3 种映射 （A pill=职位 / B hover 展开职位 / C pill=编号）。Vera：选 B 方向，但改为 姓名 + 副标题一开始就常显，hover 时背景层向上展开把文字衬得更清楚（占位用真实人像）。

R2 磨砂层高度 关键迭代 单独打开 ↗

pcard 默认磨砂层 50%，hover 上展时窜得比文字高一大截。对比 50 / 42 / 36 / 30% （每档并排「静止 / 展开」）。Vera：选 36%——盖住姓名+职位、文字落在磨砂面上更清楚，又不窜太高。

R3 图片网格 + 文字网格 混合 关键迭代 单独打开 ↗

数据形状问题：portfolio 有 79 家、却只有 8 张占位纹理、无企业实拍 / logo，全做图片卡会很长且配图重复。 选定混合版：顶部 16 家精选做图片卡（pcard V1 · 4×4），其余 63 家保持原文字网格；badge=年份，已上市副标题前加蓝色「已上市 ·」。

R4 logo 叠加 关键迭代 单独打开 ↗

在图片卡上叠企业 logo。先对比「直接叠加 / 磨砂白片」→ Vera：选直接叠加（更简洁）。 再定 logo 只在 hover 后浮现（静止已显示中文名，再放 logo 就重复），居中淡入。 深色纹理底上黑 logo 会发灰 → 落地时给精选卡优先挑浅纹理。logo 暂用来牟科技 LYMOW 统一占位。

★ 选定版（合并 R3 布局 + R4 logo） ★ 当前 单独打开 ↗

把混合布局与 hover 浮现 logo 合到一起：16 家精选图片卡（4×4）+ 63 家文字网格， 图片卡静止显示中文名 + 副标题，hover 时 logo 居中淡入 + 磨砂上展。Vera：目前很好了。

落地说明：定稿 = 混合网格（R3）+ 图片卡 hover 浮现 logo（R4，直接叠加）。届时把网格结构 / 样式精确写进 portfolio-ch.html。待补：各企业 logo（现仅 assets/logo-lymow.png 一个占位）、精选 16 家名单、 深色纹理优先换浅、真实企业配图。
