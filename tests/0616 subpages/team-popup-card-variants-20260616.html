团队成员弹窗卡片 · 迭代汇总

点 team 卡片后弹出的成员详情弹窗。从"还原参考图布局"出发，逐步引入 homepage 的 rail 分割语言、把操作图标嵌进 rail 分块、并落到浅色主题。下面按迭代顺序排列，每段可单独打开看完整交互（× / ESC / ← → 翻页 / 主题切换）。

R1 · Layout 还原 → R2 · Rail 分割 + icon 入格 + 浅/深双主题 → ★ R3 · 浅色 + Elevation
R1 Layout 还原 基础版 单独打开 ↗

照参考图 1:1 还原：左肖像 + 右深色面板；面板内「投资团队」标签置顶、× 右上、姓名（accent 竖条）+ 职位 + 简介底对齐、侧边 ←→ 翻页、底部拖拽条。配色 / 字体 / 字号全部换成我们 design system 的 token。Vera：layout 已还原，但操作图标都还浮在面板内。

R2 Rail 分割 + icon 入格 + 浅/深双主题 关键迭代 单独打开 ↗

沿卡片四条边把线延伸到视口，形成 homepage rail 那样的九宫格分割（卡片是正中格）；把 close / prev / next 嵌进 rail 分块（close 右上格、prev 左中格、next 右中格，整块可点、hover 淡填充）；新增浅 / 深主题切换（左下角小钮）。Vera：浅色布局挺好，rail 分割与 icon 入格都成立。

R3 浅色 + Elevation（当前方向） ★ 当前 单独打开 ↗

锁定浅色，专测卡片从奶白底上"浮起"的处理，顶部按钮可切：无 / 极淡描边 / 柔和投影 / 大扩散 / 蓝染投影 / 描边+投影。预览默认展示「柔和投影」。Vera：目前很好了。elevation 最终定一个即可（推荐 柔和投影 或 描边+投影，最不压过 rail 的分割语言）。
